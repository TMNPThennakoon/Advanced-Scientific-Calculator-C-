- `main-calculator.png` - Main calculator interface
- `scientific-calculator.png` - Scientific functions view
- `unit-converter.png` - Unit converter interface
- `temperature-converter.png` - Temperature converter interface
- `splash-screen.png` - Application splash screen

**Recommended image format**: PNG or JPG
**Recommended size**: 1280x720 or higher for best quality
